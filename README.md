---

## 📌 About This Project

This framework was designed and implemented at **360factors** to automate the **Predict360 GRC platform** — an AI-powered Governance, Risk & Compliance solution used by banks and financial institutions across the USA.

### Key Achievements
- ✅ 80%+ regression coverage across Predict360 risk & compliance modules
- ✅ Reduced manual regression effort by ~50% via Jenkins CI/CD integration
- ✅ BDD strategy with Cucumber enabled non-technical stakeholders to validate acceptance criteria
- ✅ Accelerated release cycles through automated pipelines
